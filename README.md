# Buch-Web

## Setup

```bash
pnpm i
pnpm dev
```

### Features

- React Framework: Remix
- Type Checking: TypeScript
- Authentication: Keycloak + Remix-Auth
- Logging: Pino
- e2e Tests: Playwright
- CSS Framework: Chakra UI
- Validation: Zod
- HTTP Client: Axios
- Icons: Lucide-React
- Formatter: Prettier
- Code Analysis: ESLint(-Plugins) + SonarQube
- Documentation: PlantUML
- Bundler: Vite
